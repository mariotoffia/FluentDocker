# Learning Path

Use this page to move through FluentDocker in layers, from simple to advanced.

## Start Here (First 30 Minutes)

1. [Install and verify prerequisites](getting-started.html#installation)
2. [Run your first container](getting-started.html#your-first-container)
3. [Add one wait strategy](getting-started.html#with-wait-strategy)
4. [Review cleanup and exception basics](getting-started.html#exception-handling)

If this is your first time with FluentDocker, finish the four steps above before opening architecture or extensibility docs.
